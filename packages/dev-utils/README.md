## Dev Utils

Dev utils to be shared across Sablier projects and packages.

## Usage

Install the module:

```bash
$ yarn add @sablier/dev-utils
```

And import it in your project:

```js
const sablierDevUtils = require("@sablier/dev-utils");
```
